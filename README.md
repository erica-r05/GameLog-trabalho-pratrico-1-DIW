# GameLog

GameLog é uma aplicação web desenvolvida como **trabalho prático da disciplina de Desenvolvimento de Interfaces Web (DIW)**.  
O projeto funciona como uma **biblioteca de jogos**, permitindo ao usuário visualizar e pesquisar informações sobre diferentes games de forma simples e organizada.

---

## Objetivo do Projeto

O principal objetivo do GameLog é aplicar, na prática, os conceitos estudados em DIW, como:

- Estruturação de páginas com HTML  
- Estilização com CSS  
- Interatividade utilizando JavaScript  
- Organização de um projeto front-end
- Simulação de um back-end

---

##  Funcionalidades

-  Listagem de jogos  
-  Pesquisa por jogos  
-  Visualização de informações básicas sobre cada game  
-  Navegação simples e intuitiva  

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Node.js / NPM (para gerenciamento de dependências)

---

## Estrutura do Projeto

- public/          → arquivos HTML, CSS e JavaScript
- db/              → base de dados ou arquivos relacionados
- node_modules/    → dependências do projeto
- package.json     → configurações e dependências
- README.md        → documentação do projeto
